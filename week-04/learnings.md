# My AI Engineering Learnings

> This file is your personal context map for AI agents. Keep it in
> `.gitignore` — it's private by default. At the start of any AI session,
> paste the contents or refer to the file and say: _"Read this file first."_
>
> Per [outcome engineering](https://o16g.com/): you can't dispatch an
> agent toward a useful destination without mapping the territory first.
> This file is that map. Add to it whenever you learn something worth
> remembering.
>
> **Tip:** You can ask your agent to help update this file. At the end
> of a session, try: _"Based on what we just did, what should I add to
> my LEARNINGS.md?"_
